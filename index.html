In retrospect
  • Mosaic, the first popular graphical Web browser
  • Netscape Communications realized that the Web needed to become more dynamic, 1993
  • They decided to develop a scripting language, easy to use, where the code could be written directly in the Web page markup
  • Brenden Eich
In retrospect
  • Later in 96, JScript, a reverse-engineered implementation of Netscape's JavaScript, was part of Internet Explorer 3
  • Internet Explorer 3 also included Microsoft's first support for CSS and various extensions         to HTML, but in each case the implementation was noticeably different to that found in         Netscape Navigator at the time
  • Browser wars. Standardization was needed
  • Netscape submitted JavaScript to Ecma International to carve out a standard specification
In retrospect
  • Ajax, Jesse James
  • web applications where data can be loaded in the background, avoiding the need for full         page reloads and leading to more dynamic applications
  • Prototype, jQuery, Dojo Toolkit, MooTools
  • SPA. Angular, Backbone, React , Ember ...

Angular

Open Source Javascript Framework

  • get rid of difficulty in maintaining initial codebase
  • Code Production became fast
  • Developed by Google
  • Easily testable
  • Actively supported by Communities

Key Features

  • easy data binding,two way data binding
  • Reusable Components, Directives
  • MVC Design pattern
  • Routing
  • Templating

How

  • Scope is the glue between application controller and the view
  • As the scope grows affects performance
  • Digest Cycle, Dirty Checking
  • Tightly coupled entities like controllers, views, services
  • EvolvingWeb

Angular2

  • Everything is a Component
  • No Scope, digest cycle No controller in Angular2
  • Better Dependency Injection through Constructor
  • Loosely coupled
  • Typescript
  • Faster
